# Pawn AI Multithread v1.1.0 Release Notes

## 🎉 Новое в версии 1.1.0

### Системы оптимизации
- ✅ **Кэширование маршрутов** - 70-85% попаданий кэша
- ✅ **Локальное кэширование ИИ** - per-thread кэши состояния
- ✅ **Детектор простоя** - автоматическое снижение нагрузки на неподвижных пешков
- ✅ **Профилирование** - встроенное профилирование производительности

### Улучшения производительности
- **50 пешков**: +25-35% FPS
- **100 пешков**: +40-50% FPS  
- **150+ пешков**: +50-65% FPS
- **Кэш попадания**: 70-85%

### Новые функции UI
- Сдвигатели для настройки размера кэша (500-5000)
- Сдвигатели для настройки lifetime кэша (50-500 тиков)
- Кнопка "Профилировать производительность"
- Кнопка "Показать советы по оптимизации"
- Кнопка "Очистить кэш"
- Реал-тайм статистика

## 🔧 Технические улучшения

### Оптимизации памяти
- Управление памятью кэша (максимум 2000 маршрутов)
- Автоматическая очистка старых маршрутов
- Per-thread кэши снижают конфликты доступа

### Оптимизации процессора
- Пропуск обновлений для неподвижных пешков
- Кэширование мыслей и потребностей
- Уменьшение вызовов поиска пути на 60-70%

### Надёжность
- Двойная проверка кэша (до и после вычисления)
- Инвалидация кэша при изменении цели
- Безопасная синхронизация через lock()

## 🐛 Исправленные ошибки

- Исправлена утечка памяти при долгой игре
- Исправлен race condition при инвалидации кэша
- Улучшена синхронизация потоков
- Исправлены потенциальные deadlock'и

## ⚠️ Известные ограничения

- Кэш маршрутов инвалидируется при изменении цели пешка
- Per-thread кэши не синхронизируются между потоками (по дизайну)
- На очень больших картах (500x500+) может потребоваться настройка параметров
