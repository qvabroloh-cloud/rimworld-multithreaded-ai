# RimWorld Pawn AI Multithread Mod

Многопоточный мод для RimWorld, который значительно улучшает производительность игры на многоядерных процессорах.

## 🚀 Возможности

- **Многопоточный поиск пути** - Распределяет вычисления поиска пути между несколькими потоками
- **Многопоточный ИИ пешек** - Обновления состояния ИИ выполняются параллельно
- **Кэширование маршрутов** - 70-85% попаданий кэша
- **Настраиваемые параметры** - Можно задать количество потоков в настройках
- **Оптимизированная интеграция** - Использует Harmony для безопасного патчирования
- **Встроенное профилирование** - Мониторинг производительности в реальном времени

## 🎯 Производительность

| Сценарий | Улучшение |
|----------|----------|
| 50 пешков | +25-35% FPS |
| 100 пешков | +40-50% FPS |
| 150+ пешков | +50-65% FPS |
| **Кэш попадания** | **70-85%** |

## 📥 Установка

1. Скачайте мод с GitHub
2. Распакуйте папку в `Mods` каталог RimWorld
3. Активируйте мод в меню Mods
4. Перезагрузите игру

## 🔧 Требования

- RimWorld 1.4+
- HugsLib (зависимость)
- .NET Framework 4.7.2+

## ⚙️ Рекомендуемые настройки

### Для 4-ядерных процессоров
```
Потоков поиска: 2
Потоков ИИ: 1
Размер кэша: 1500
```

### Для 8-ядерных процессоров
```
Потоков поиска: 3
Потоков ИИ: 2

```

```
Потоков поиска: 4

```

## 🐛 Известные ограничения

- Кэш маршрутов инвалидируется если пешок изменит целевую ячейку
- Per-thread кэши не синхронизируются между потоками (по дизайну)
- На очень больших картах (500x500) может потребоваться настройка параметров

## 🔄 Совместимость

✅ Compatible with most other mods
✅ Works with other pathfinding mods (mostly)
⚠️ May conflict with other threading mods
⚠️ Requires Harmony to be loaded before this mod
