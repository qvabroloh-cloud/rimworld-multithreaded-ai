# RimWorld Pawn AI Multithread - Быстрая сборка

## 🚀 Как использовать скрипты сборки

### Вариант 1: ПОЛНАЯ АВТОМАТИЗАЦИЯ (рекомендуется)

**Просто нажмите на файл:**
```
setup.bat
```

Он автоматически:
1. Скомпилирует весь код
2. Найдёт RimWorld
3. Установит мод в игру
4. Готово!

### Вариант 2: Отдельно компилировать и устанавливать

**Сначала компилируйте:**
```
build.bat
```
Это создаст файл: `Source/bin/Release/PawnAIMultithread.dll`

**Потом устанавливайте:**
```
install.bat
```
Это скопирует мод в RimWorld

### Вариант 3: Ручная компиляция

Если скрипты не работают:
```
msbuild Source\PawnAIMultithread.csproj /p:Configuration=Release
```

---

## ⚠️ Если возникают ошибки

### Ошибка: "MSBuild не найден"
- Установите Visual Studio Community
- Выберите ".NET desktop development"
- Перезагрузитесь
- Запустите скрипт заново

### Ошибка: "RimWorld не найдена"
- Отредактируйте `install.bat`
- Найдите строку: `set RIMWORLD_PATH=`
- Измените на ваш путь к RimWorld
- Сохраните и запустите заново

### Ошибка: "DLL файл не создан"
- Проверьте, что все файлы на месте
- Убедитесь, что Visual Studio установлена правильно
- Очистите папку: `Source/bin/`
- Запустите `build.bat` заново

---

## 📋 Что делают скрипты

| Скрипт | Что делает |
|--------|----------|
| `setup.bat` | Всё в одном: компиляция + установка |
| `build.bat` | Только компиляция Code → DLL |
| `install.bat` | Только установка DLL → RimWorld |

## ✅ После установки

1. Откройте RimWorld
2. Главное меню → Mods → Manage Mods
3. Найдите "Pawn AI Multithread"
4. Поставьте галочку
5. Закройте меню
6. Создайте новую игру

**Готово! Мод активен!** 🎮
